Частина 1. Теоретичні питання
1. Що таке інформаційна система? Назвіть її основні компоненти.
Відповідь:
Інформаційна система (ІС) – це сукупність апаратного, програмного забезпечення, даних, мереж і людей, які взаємодіють для збору, обробки, зберігання та поширення інформації з метою підтримки прийняття рішень.
Основні компоненти ІС:
Апаратне забезпечення (Hardware): сервери, комп’ютери, мережеве обладнання.
Програмне забезпечення (Software): ОС, СУБД, спеціалізовані додатки.
Дані: структуровані (бази даних) та неструктуровані (файли, зображення).
Мережі: інтернет, локальні мережі (LAN), Wi-Fi.
Люди: користувачі, адміністратори, аналітики.

2. Класифікація інформаційних систем за рівнями управління
Відповідь:
Операційний рівень:
Призначення: обробка повсякденних транзакцій (наприклад, продажі, логістика).
Приклади: POS-системи, облік складських запасів.
Рівень управління знаннями:
Призначення: підтримка спеціалістів (інженери, лікарі).
Приклади: CAD-системи, медичні бази даних.
Тактичний рівень (Management Information Systems, MIS):
Призначення: аналіз даних для середнього менеджменту.
Приклади: звіти про продажі, KPI-аналітика.
Стратегічний рівень (Executive Information Systems, EIS):
Призначення: довгострокове планування для топ-менеджменту.
Приклади: прогнози ринку, BI-інструменти (Power BI, Tableau).

3. Основні переваги застосування Інтернет-технологій у бізнес-процесах
Відповідь:
Глобальна доступність: можливість працювати з клієнтами та партнерами з усього світу.
Зниження витрат: електронний документообіг, хмарні сервіси замість власних серверів.
Швидкість комунікації: миттєвий обмін даними через email, месенджери, CRM-системи.
Автоматизація: веб-додатки для обробки замовлень, чат-боти для підтримки клієнтів.
Аналітика даних: інструменти на кшталт Google Analytics для відстеження ефективності маркетингу.

5. Призначення протоколу HTTP та його роль у веб-додатках
Відповідь:
HTTP (HyperText Transfer Protocol) – це протокол для передачі даних у вебі. Він визначає формат запитів (Request) і відповідей (Response) між клієнтом (браузером) і сервером.
Роль у веб-додатках:
Користувач відкриває сайт → браузер відправляє HTTP-запит (GET /index.html).
Сервер обробляє запит і повертає HTTP-відповідь (HTML-код сторінки).
Для безпеки використовується HTTPS (зашифрована версія HTTP).
Приклад: Коли ви вводите URL у браузері, він виконує HTTP-запит, щоб завантажити сторінку.

Частина 3. Тестові завдання
6. Вкажіть правильні відповіді:
А) HTML — це мова програмування – Неправильно. HTML – це мова розмітки (Markup Language).
Б) Клієнт-серверна архітектура включає веб-сервер – Правильно. Наприклад, веб-сайти працюють за цією моделлю.
В) SQL використовується для обробки мультимедійного контенту – Неправильно. SQL – мова для роботи з базами даних.
Г) Інформаційна система обов’язково має інтерфейс користувача – Неправильно. Деякі ІС (наприклад, бекенд-API) взаємодіють лише з іншими системами.
Правильна відповідь: Б.
